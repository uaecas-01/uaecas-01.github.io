수학

개념은 아는데 왜 문제가 안 풀릴까?

⏱ 5분 읽기📅 2026.04

수학 시험을 준비하면서 교과서 개념을 분명히 이해했다고 생각했는데, 막상 문제를 풀면 손이 멈추는 경험을 해본 학생이 많습니다. 개념을 '아는 것'과 문제에 '적용하는 것' 사이에는 생각보다 큰 간극이 존재합니다. 이 글에서는 그 갭이 왜 생기는지 원인을 분석하고, 실전에서 개념을 자유자재로 활용할 수 있도록 연결 훈련법을 단계별로 안내합니다.

개념 이해와 문제 풀이 사이의 갭은 왜 생길까?

인지심리학에서는 지식을 '선언적 지식(declarative knowledge)'과 '절차적 지식(procedural knowledge)'으로 구분합니다. 교과서에서 공식과 정의를 읽고 이해하는 것은 선언적 지식에 해당합니다. 반면 문제 풀이는 절차적 지식, 즉 '어떤 상황에서 어떤 도구를 꺼내 쓸 것인가'를 판단하는 능력입니다.

예를 들어 이차방정식의 근의 공식을 외우고 있더라도, 문제에서 "두 근의 합과 곱이 주어졌을 때 원래 이차방정식을 구하라"고 하면 근과 계수의 관계를 떠올려야 합니다. 이처럼 개념을 문제 상황에 매칭하는 훈련이 부족하면 개념은 알지만 문제는 못 푸는 현상이 반복됩니다.

대표적인 원인 3가지

  1. 개념의 피상적 이해: 공식을 외웠지만 그 공식이 왜 성립하는지, 어떤 조건에서 사용하는지를 모릅니다. 예컨대 코사인 법칙을 알면서도 어떤 삼각형 조건에서 코사인 법칙이 필요한지 판단하지 못하는 경우입니다.
  2. 문제 유형 분류 경험 부족: 같은 개념이라도 문제는 다양한 형태로 출제됩니다. 충분히 다양한 유형을 접해보지 않으면 새로운 형태의 문제에서 개념을 연결하지 못합니다.
  3. 풀이 전략 수립 훈련 미비: 문제를 읽고 바로 풀이에 돌입하는 습관이 있으면, 어떤 개념을 사용해야 하는지 전략을 세우는 단계를 건너뛰게 됩니다.

1단계: 개념을 '설명할 수 있는 수준'으로 끌어올리기

단순히 공식을 읽고 넘어가는 것이 아니라, 그 개념을 다른 사람에게 설명할 수 있는 수준으로 이해해야 합니다. 이를 인지과학에서는 '정교화(elaboration)' 전략이라고 합니다.

  • 자기 설명법(Self-explanation): 교과서를 읽을 때 각 단계마다 "왜 이렇게 되지?"라고 스스로 질문하고 답합니다. 연구에 따르면 자기 설명을 하는 학생은 그렇지 않은 학생보다 문제 풀이 정확도가 평균 30% 이상 높습니다.
  • 빈 종이 테스트: 교과서를 덮고 빈 종이에 해당 단원의 핵심 개념, 공식, 그리고 각 공식의 사용 조건을 적어봅니다. 적지 못하는 부분이 실제로 이해하지 못한 부분입니다.
  • 연결 지도 만들기: 하나의 단원 안에서 개념들이 어떻게 연결되는지 마인드맵 형태로 정리합니다. 예를 들어, '이차함수 → 꼭짓점 → 축의 방정식 → 판별식 → 근의 공식'처럼 흐름을 시각화합니다.
💡 실전 팁: 개념 설명을 녹음해서 다시 들어보세요. 설명이 막히는 지점이 정확히 이해가 부족한 지점입니다. 이 방법은 특히 혼자 공부하는 학생에게 매우 효과적입니다.

2단계: 문제 유형 분류 훈련하기

같은 개념이라도 문제의 출제 방식은 다양합니다. 개념을 완전히 이해한 후에는 해당 개념이 어떤 유형의 문제로 변형되는지를 파악하는 훈련이 필요합니다.

유형 분류 방법

  1. 문제 모으기: 하나의 개념에 대해 교과서, 문제집, 기출에서 관련 문제를 10개 이상 모읍니다.
  2. 출제 패턴 분석: 모은 문제들을 "직접 대입형", "조건 추론형", "그래프 해석형", "증명·서술형" 등으로 분류합니다.
  3. 패턴별 풀이 전략 정리: 각 유형마다 첫 번째로 해야 할 행동을 명확히 적습니다. 예를 들어 "조건 추론형이면 주어진 조건을 수식으로 변환하는 것부터 시작한다"처럼 구체적으로 정리합니다.
⚠️ 주의: 유형 분류는 기계적 암기가 아닙니다. "이런 유형이면 이 공식"이라고 외우는 것이 아니라, "이런 조건이 주어지면 이 개념이 필요하겠다"라고 판단하는 사고 과정을 훈련하는 것입니다. 수능에서는 단순 유형 암기로는 풀 수 없는 융합 문제가 출제됩니다.

3단계: 문제 읽기 전략 바꾸기

많은 학생이 문제를 읽자마자 바로 풀이를 시작합니다. 하지만 상위권 학생들은 문제를 읽는 단계에서 이미 풀이 전략을 수립합니다.

  • 조건 밑줄 긋기: 문제에서 주어진 조건에 밑줄을 긋고, 각 조건이 어떤 개념과 연결되는지 옆에 메모합니다.
  • 구하는 것 명확히 하기: "결국 이 문제가 요구하는 것은 무엇인가?"를 한 문장으로 정리합니다. 의외로 많은 학생이 문제가 무엇을 묻고 있는지를 정확히 파악하지 못한 채 풀이를 시작합니다.
  • 풀이 로드맵 구상: 실제 계산에 들어가기 전에 "조건 정리 → 사용할 개념 → 예상 풀이 단계"를 30초간 머릿속으로 그려봅니다.

4단계: 오답에서 '연결 실패 지점' 찾기

틀린 문제를 분석할 때 단순히 "계산 실수"로 분류하지 마세요. 대부분의 오답은 개념과 문제 사이의 연결이 끊어진 지점에서 발생합니다.

  • 오답 원인 세분화: 틀린 문제마다 다음 중 어디에서 막혔는지를 기록합니다. (1) 어떤 개념을 써야 하는지 몰랐다 (2) 개념은 떠올렸지만 적용 방법을 몰랐다 (3) 적용은 했지만 계산 과정에서 실수했다 (4) 조건을 잘못 해석했다.
  • 패턴 발견: 오답 원인을 2주 이상 기록하면 자신만의 취약 패턴이 보입니다. (1)번이 많다면 개념 정리가 부족한 것이고, (2)번이 많다면 유형 훈련이 필요한 것입니다.

5단계: 간격 반복으로 연결 강화하기

한 번 연결에 성공한 개념-문제 쌍도 시간이 지나면 다시 끊어질 수 있습니다. 에빙하우스의 망각 곡선에 따르면, 학습 후 24시간 이내에 복습하지 않으면 70% 이상을 잊습니다. 따라서 간격을 두고 반복하는 전략이 필수적입니다.

  • 1일 후: 틀린 문제를 다시 풀어봅니다.
  • 3일 후: 같은 개념의 다른 유형 문제를 풀어봅니다.
  • 7일 후: 해당 개념이 포함된 종합 문제를 풀어봅니다.
  • 14일 후: 모의고사 형태의 실전 문제에서 해당 개념을 적용해봅니다.
💡 핵심 정리: 개념과 문제 풀이의 갭을 메우려면 "깊이 있는 이해 → 유형 분류 → 전략적 문제 읽기 → 오답 연결 분석 → 간격 반복"의 5단계 훈련이 필요합니다. 단순히 문제를 많이 푸는 것이 아니라, 개념이 문제 속에서 어떻게 변형되는지를 의식적으로 추적하는 습관이 핵심입니다.
