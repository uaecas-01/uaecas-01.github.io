중간고사 2주 전, 뭐부터 해야 할까?

⏱ 4분 읽기📅 2026.04

중간고사가 2주 앞으로 다가왔을 때, 많은 학생들이 "어디서부터 시작해야 할지 모르겠다"고 말합니다. 시험 범위는 넓고 시간은 부족하게 느껴지지만, 체계적인 계획만 세우면 2주는 충분히 의미 있는 준비 기간이 됩니다. 이 글에서는 실제 성적 상위권 학생들이 활용하는 2주 플래닝 전략을 단계별로 안내합니다.

1단계: 시험 범위와 현재 상태 파악하기 (D-14~13)

가장 먼저 해야 할 일은 각 과목의 시험 범위를 정확하게 확인하는 것입니다. 교과서 페이지 수, 프린트 자료, 수업 중 필기 내용을 모두 모아 과목별로 정리합니다.

  • 범위 체크리스트 작성: 과목별로 시험 범위에 해당하는 교과서 단원, 부교재, 프린트를 나열합니다. 빠뜨린 자료가 없는지 친구에게 확인합니다.
  • 자가 진단 테스트: 각 과목의 핵심 개념 10개를 적고, 설명할 수 있는지 체크합니다. 설명이 되지 않는 개념은 우선 학습 대상입니다.
  • 과목별 난이도 등급 분류: 본인 기준으로 A(자신 있음), B(보통), C(취약) 등급으로 분류합니다. C등급 과목에 전체 학습 시간의 40% 이상을 배분해야 합니다.

2단계: 과목별 우선순위와 시간 배분 (D-12~10)

모든 과목에 동일한 시간을 투자하는 것은 비효율적입니다. 성적을 가장 빠르게 올릴 수 있는 과목부터 집중하는 전략이 필요합니다.

우선순위 결정 기준

  1. 현재 점수 대비 상승 여지: 70점대 과목을 90점으로 올리는 것이 95점을 100점으로 올리는 것보다 투자 대비 효과가 큽니다.
  2. 암기 비중이 높은 과목 먼저: 사회, 과학 같은 암기 과목은 직전에 집중해도 효과가 크지만, 수학처럼 이해가 필요한 과목은 먼저 시작해야 합니다.
  3. 내신 반영 비율 고려: 대입에서 내신 반영 비율이 높은 과목이 있다면 해당 과목에 가중치를 두세요.
💡 실전 팁: 하루를 3개 시간 블록(아침/오후/저녁)으로 나누고, 각 블록마다 다른 과목을 배치하세요. 같은 과목을 3시간 연속으로 하는 것보다 1.5시간씩 나눠서 하는 것이 집중력 유지에 훨씬 효과적입니다.

3단계: 1차 학습 – 개념 정리 (D-9~7)

이 기간에는 교과서와 수업 필기를 중심으로 개념을 다시 잡습니다. 단순히 읽는 것이 아니라, 핵심 키워드를 뽑아 정리 노트를 만드는 것이 중요합니다.

  • 키워드 추출법: 각 단원에서 굵은 글씨, 정의, 공식, 그래프를 모두 별도 노트에 옮겨 적습니다.
  • 선생님 수업 필기 재정리: 수업 중 선생님이 "이거 시험에 나와요"라고 말한 부분을 형광펜으로 표시하고, 해당 내용을 확실히 이해합니다.
  • 개념 확인 문제 풀기: 교과서 끝에 있는 확인 문제나 기본 문제집의 개념 확인 문제를 풀어봅니다. 틀린 문제는 해당 개념으로 돌아가 복습합니다.

4단계: 2차 학습 – 문제 풀이 (D-6~4)

개념 정리가 끝났으면 본격적으로 문제를 풀어야 합니다. 이 단계에서는 기출문제와 예상 문제를 중심으로 실전 감각을 키웁니다.

  • 기출문제 분석: 이전 시험 기출을 구할 수 있다면 반드시 풀어봅니다. 같은 선생님이라면 출제 패턴이 비슷한 경우가 많습니다.
  • 오답 노트 작성: 틀린 문제마다 왜 틀렸는지(개념 부족/계산 실수/문제 독해 실패) 원인을 분류합니다.
  • 변형 문제 연습: 같은 개념이지만 다른 형태로 출제된 문제를 풀어 응용력을 기릅니다.
⚠️ 주의: 문제를 많이 푸는 것보다 틀린 문제를 확실히 이해하는 것이 더 중요합니다. 하루에 100문제를 풀고 넘기는 것보다, 50문제를 풀고 틀린 20문제를 완벽히 분석하는 것이 점수 향상에 훨씬 효과적입니다.

5단계: 최종 점검 – 마무리 복습 (D-3~1)

시험 직전 3일은 새로운 내용을 공부하기보다 이미 학습한 내용을 반복 확인하는 시간입니다.

  • D-3: 정리 노트와 오답 노트를 처음부터 끝까지 읽으며 빈칸이 있는 부분을 보완합니다.
  • D-2: 각 과목의 핵심 공식, 정의, 키워드를 A4 한 장에 요약합니다. 이것이 시험 직전 최종 리뷰 자료가 됩니다.
  • D-1: 실제 시험 시간에 맞춰 모의 시험을 치르거나, A4 요약 시트를 반복해서 읽습니다. 밤샘은 절대 금지입니다. 7시간 이상 수면이 기억 정착의 핵심입니다.

시험 당일 체크리스트

  • 기상 후 가벼운 스트레칭과 아침 식사(탄수화물 + 단백질 조합 권장)
  • A4 요약 시트로 15분간 최종 리뷰
  • 시험 시작 전 심호흡 3회로 긴장 완화
  • 시험지를 받으면 전체를 한 번 훑어보고, 쉬운 문제부터 풀기 시작
  • 서술형은 채점 기준 키워드를 반드시 포함하여 작성
💡 핵심 정리: 2주 플래닝의 핵심은 "범위 파악 → 우선순위 결정 → 개념 정리 → 문제 풀이 → 최종 복습" 5단계입니다. 매일 계획 대비 실행률을 체크하고, 미달인 부분은 다음 날로 이월하되 과목별 최소 학습량은 반드시 지키세요.
