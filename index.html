왜 WeStudy인가요?

차별화된 학습 코칭
시스템의 비밀

WeStudy는 단순 매칭이 아닌, 아이의 성장에 진심으로 관심을 갖는 파트너입니다.

✓

데이터 기반 선생님 매칭
학생의 성취도 데이터와 선생님의 수업 이력을 분석하여 최적의 조합을 찾습니다.

✓

주 1회 학습 리포트 제공
수업 내용, 진도, 취약점 분석을 포함한 상세 리포트를 매주 학부모님께 공유합니다.

✓

불만족 시 선생님 교체 보장
수업 방향이 맞지 않을 경우 추가 비용 없이 즉시 선생님을 교체해 드립니다.

✓

목동 지역 집중 네트워크
목동, 신정동, 양천구 지역의 탄탄한 학원·선생님 네트워크를 보유하고 있습니다.
